HTML Details & Summary Elements

The <details> and <summary> elements create collapsible content sections that users can expand and collapse by clicking. This is perfect for FAQs, accordions, and progressive disclosure.

Basic Details & Summary

Simple Collapsible Content

HTML Code:
<details>
  <summary>Click to expand</summary>
  <p>This content is hidden by default and shows when you click the summary.</p>
  <p>You can put any HTML content here - text, images, lists, etc.</p>
</details>

HTML Progress & Meter Elements

The <progress> element shows the completion progress of a task, while <meter> displays a scalar value within a known range (like disk usage or temperature).

Progress Element

Progress Bars

HTML Code:
<!-- Progress with value and max -->
<label for="file-progress">File upload progress:</label>
<progress id="file-progress" value="70" max="100">70%</progress>

<!-- Progress without value (indeterminate) -->
<label for="loading-progress">Loading:</label>
<progress id="loading-progress">Loading...</progress>

<!-- Different progress values -->
<p>Download: <progress value="25" max="100">25%</progress> 25%</p>
<p>Installation: <progress value="80" max="100">80%</progress> 80%</p>
<p>Complete: <progress value="100" max="100">100%</progress> 100%</p>

Meter Element

Gauges and Measurements

HTML Code:
<!-- Basic meter -->
<label for="disk-usage">Disk usage:</label>
<meter id="disk-usage" value="6" min="0" max="10">6 out of 10</meter>

<!-- Meter with optimum value -->
<label for="battery">Battery level:</label>
<meter id="battery" value="85" min="0" max="100" optimum="80">85%</meter>

<!-- Meter with low/high thresholds -->
<label for="temperature">Temperature:</label>
<meter id="temperature" value="75" min="0" max="100" low="20" high="80" optimum="70">75¬∞F</meter>

HTML Iframe Element

The <iframe> element embeds another HTML document within the current document. It's commonly used for embedding videos, maps, social media content, and other external resources.

Basic Iframe Usage

Embedding External Content

HTML Code:
<!-- Basic iframe -->
<iframe src="https://example.com" width="600" height="400">
  Your browser does not support iframes.
</iframe>

<!-- Iframe with title for accessibility -->
<iframe src="https://example.com"
        width="600"
        height="400"
        title="Example website">
  Your browser does not support iframes.
</iframe>

<!-- Responsive iframe -->
<iframe src="https://example.com"
        width="100%"
        height="300"
        title="Responsive iframe">
  Your browser does not support iframes.
</iframe>

Embedding YouTube Videos

HTML Code:
<!-- YouTube video embed -->
<iframe width="560"
        height="315"
        src="https://www.youtube.com/embed/VIDEO_ID"
        title="YouTube video player"
        frameborder="0"
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
        allowfullscreen>

<div style="position: relative; width: 100%; height: 0; padding-bottom: 56.25%;">

          frameborder="0"

Iframe Attributes & Security

Important Iframe Attributes

HTML Code:
<!-- Iframe with security attributes -->
<iframe src="https://example.com"
        width="600"
        height="400"
        title="Example website"
        frameborder="0"
        sandbox="allow-scripts allow-same-origin"
        loading="lazy"
        referrerpolicy="no-referrer">
  Your browser does not support iframes.
</iframe>

<!-- Google Maps embed -->
<iframe src="https://www.google.com/maps/embed?pb=..."
        width="600"
        height="450"
        style="border:0;"
        allowfullscreen=""
        loading="lazy"
        referrerpolicy="no-referrer-when-downgrade"
        title="Google Maps">
</iframe>

  • ‚Ä¢ loading="lazy": Improves page performance
  • ‚Ä¢ title: Required for accessibility
  • ‚Ä¢ Only embed trusted content

Key Takeaways

  • HTML provides interactive elements that work without JavaScript
  • Details and summary create accessible collapsible content
  • Progress elements show task completion, meter elements show measurements
  • Iframes embed external content but require security considerations
  • Always include proper accessibility attributes and fallback content
  • These elements enhance user experience with minimal code
  • Test interactive elements across different browsers and devices
  • Consider performance implications when embedding external content
