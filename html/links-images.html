Introduction to Links and Images

Links and images are fundamental elements that make the web interactive and visually appealing. Links connect pages together, creating the "web" of interconnected content, while images enhance user experience and convey information visually.

HTML Links (Anchor Tags)

The <a> (anchor) element creates hyperlinks. The href attribute specifies the destination:

Basic Link Syntax:

<a href="destination">Link Text</a>

Types of Links

HTML Code:

<!-- External Links -->
<a href="https://www.google.com">Visit Google</a>
<a href="https://www.youtube.com" target="_blank">YouTube (New Tab)</a>

<!-- Internal Links (same website) -->
<a href="about.html">About Page</a>
<a href="../index.html">Home Page</a>

<!-- Anchor Links (same page) -->
<a href="#section1">Go to Section 1</a>
<a href="#top">Back to Top</a>

<!-- Email Links -->
<a href="mailto:contact@example.com">Send Email</a>
<a href="mailto:info@example.com?subject=Hello">Email with Subject</a>

<!-- Phone Links -->
<a href="tel:+1234567890">Call Us</a>

Result:

Visit Google

YouTube (New Tab)

About Page

Send Email

Call Us

Link Attributes

target="_blank"

Opens link in a new tab/window

title="Description"

Shows tooltip on hover

rel="noopener"

Security attribute for external links

HTML Images

The <img> element embeds images in web pages. It's a self-closing tag with required attributes:

Basic Image Syntax:

<img src="image-path" alt="description">

Image Examples

HTML Code:

<!-- Basic Image -->
<img src="logo.png" alt="Company Logo">

<!-- Image with Size -->
<img src="photo.jpg" alt="Beautiful sunset" width="300" height="200">

<!-- Responsive Image -->
<img src="banner.jpg" alt="Website banner" style="max-width: 100%; height: auto;">

<!-- Image with Title -->
<img src="icon.png" alt="Settings icon" title="Click to open settings">

<!-- Placeholder Image -->
<img src="https://placehold.co/300x200/orange/white?text=Sample+Image" 
     alt="Sample placeholder image">

Result:

Sample image with orange background

Image Attributes

src (required)

Path to the image file

alt (required)

Alternative text for accessibility and SEO

width & height

Dimensions in pixels (use CSS instead when possible)

title

Tooltip text that appears on hover

Combining Links and Images

You can make images clickable by wrapping them in anchor tags:

HTML Code:

<!-- Clickable Image -->
<a href="https://www.example.com">
  <img src="button.png" alt="Click here to visit our website">
</a>

<!-- Image Link with Caption -->
<figure>
  <a href="gallery.html">
    <img src="thumbnail.jpg" alt="Photo gallery thumbnail">
  </a>
  <figcaption>Click to view full gallery</figcaption>
</figure>

File Paths and Organization

üìÅ File Path Types:

Relative paths:

  • image.jpg - Same folder
  • images/photo.jpg - Subfolder
  • ../images/logo.png - Parent folder

Absolute paths:

  • https://example.com/image.jpg - External URL
  • /images/banner.jpg - Root-relative

Best Practices

‚úÖ Do

  • Always include alt text for images
  • Use descriptive link text
  • Optimize image file sizes
  • Use appropriate image formats
  • Test all links regularly
  • Use rel="noopener" for external links

‚ùå Don't

  • Use "click here" as link text
  • Leave alt attributes empty
  • Use images for text content
  • Link to broken or missing pages
  • Use huge unoptimized images
  • Forget to test mobile responsiveness

Try It Yourself

Exercise:

Create a simple photo gallery page:

  1. Create a page with a main heading
  2. Add 3-4 placeholder images using placehold.co
  3. Make each image clickable (link to a larger version)
  4. Add proper alt text for accessibility
  5. Include navigation links to other pages
  6. Add your email contact link

Complete Example

Photo Gallery Page:

<!DOCTYPE html>
<html lang="en">
<head>
  <title>My Photo Gallery</title>
</head>
<body>
  <nav>
    <a href="index.html">Home</a> |
    <a href="about.html">About</a> |
    <a href="contact.html">Contact</a>
  </nav>
  
  <h1>My Photo Gallery</h1>
  
  <div class="gallery">
    <figure>
      <a href="images/sunset-large.jpg">
        <img src="images/sunset-thumb.jpg" 
             alt="Beautiful sunset over the ocean">
      </a>
      <figcaption>Sunset at the beach</figcaption>
    </figure>
    
    <figure>
      <a href="images/mountain-large.jpg">
        <img src="images/mountain-thumb.jpg" 
             alt="Snow-capped mountain peak">
      </a>
      <figcaption>Mountain adventure</figcaption>
    </figure>
  </div>
  
  <footer>
    <p>Contact me: <a href="mailto:photographer@example.com">
       photographer@example.com</a></p>
  </footer>
</body>
</html>

Key Takeaways

  • Use <a> tags with href attribute to create links
  • target="_blank" opens links in new tabs
  • <img> tags require src and alt attributes
  • Alt text is crucial for accessibility and SEO
  • Images can be made clickable by wrapping in <a> tags
  • Use relative paths for internal resources
  • Always test links and optimize image sizes
