üëã Personal Introduction Page

Create a webpage that introduces yourself to the world

üìù Project Requirements

Page Structure
  • ‚Ä¢ Proper HTML5 document structure
  • ‚Ä¢ Meaningful page title
  • ‚Ä¢ Meta tags for charset and viewport
  • ‚Ä¢ Semantic HTML elements
Content Requirements
  • ‚Ä¢ Main heading with your name
  • ‚Ä¢ Introduction paragraph about yourself
  • ‚Ä¢ Your hobbies or interests (use a list)
  • ‚Ä¢ Contact information section
  • ‚Ä¢ Fun fact about yourself
HTML Elements to Use
  • ‚Ä¢ h1, h2, h3 for headings
  • ‚Ä¢ p for paragraphs
  • ‚Ä¢ strong, em for emphasis
  • ‚Ä¢ ul or ol for lists
  • ‚Ä¢ HTML comments for documentation

üéØ Step-by-Step Guide

Step 1: Set up HTML structure

Create the basic HTML5 document with DOCTYPE, html, head, and body tags

Step 2: Add meta information

Include title, charset, and viewport meta tags in the head section

Step 3: Create main heading

Add an h1 tag with your name as the main page heading

Step 4: Write introduction

Add 2-3 paragraphs introducing yourself, your background, and goals

Step 5: Add interests section

Create an h2 heading and list your hobbies using ul and li tags

Step 6: Include contact info

Add another section with your contact information (email, social media)

Step 7: Add documentation

Include HTML comments to document different sections of your page

üí° Example Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Name - Personal Introduction</title>
</head>
<body>
    <!-- Main header section -->
    <header>
        <h1>Hello, I'm [Your Name]!</h1>
    </header>

    <!-- Main content -->
    <main>
        <section>
            <h2>About Me</h2>
            <p>Write your introduction here...</p>
        </section>

        <section>
            <h2>My Interests</h2>
            <ul>
                <li>Interest 1</li>
                <li>Interest 2</li>
                <li>Interest 3</li>
            </ul>
        </section>
    </main>
</body>
</html>

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Chocolate Chip Cookies - My Recipe Collection</title>
</head>
<body>
    <header>
        <h1>My Recipe Collection</h1>
        <nav>
            <ul>
                <li><a href="index.html">Home</a></li>
                <li><a href="cookies.html">Cookies</a></li>
                <li><a href="pasta.html">Pasta</a></li>
                <li><a href="salad.html">Salad</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <article>
            <h2>Chocolate Chip Cookies</h2>
            <p><strong>Prep Time:</strong> 15 minutes</p>
            <p><strong>Cook Time:</strong> 12 minutes</p>
            <p><strong>Servings:</strong> 24 cookies</p>

            <h3>Ingredients</h3>
            <ul>
                <li>2&frac14; cups all-purpose flour</li>
                <li>1 cup butter, softened</li>
                <li>&frac34; cup brown sugar</li>
                <li>2 large eggs</li>
                <li>2 cups chocolate chips</li>
            </ul>

            <h3>Instructions</h3>
            <ol>
                <li>Preheat oven to 375&deg;F.</li>
                <li>Mix butter and sugars until creamy.</li>
                <li>Add eggs and vanilla; mix well.</li>
                <li>Gradually add flour mixture.</li>
                <li>Stir in chocolate chips.</li>
                <li>Bake for 9-11 minutes until golden.</li>
            </ol>
        </article>
    </main>
</body>
</html>
