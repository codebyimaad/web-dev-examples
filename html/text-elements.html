Introduction to Text Elements

Text is the foundation of web content. HTML provides various elements to structure and format text, from headings and paragraphs to emphasis and special formatting. Understanding these elements is crucial for creating well-structured, accessible web pages.

HTML Headings

HTML provides six levels of headings, from <h1> (most important) to <h6> (least important):

HTML Code:

<h1>Main Heading (Most Important)</h1>
<h2>Section Heading</h2>
<h3>Subsection Heading</h3>
<h4>Sub-subsection Heading</h4>
<h5>Minor Heading</h5>
<h6>Smallest Heading</h6>

üí° Best Practices:

  • Use only one <h1> per page (for SEO)
  • Don't skip heading levels (h1 ‚Üí h2 ‚Üí h3, not h1 ‚Üí h3)
  • Use headings for structure, not just styling

Paragraphs and Line Breaks

The <p> element defines paragraphs, while <br> creates line breaks:

HTML Code:

<p>This is the first paragraph. It contains multiple sentences 
that form a complete thought or idea.</p>

<p>This is the second paragraph. Notice how paragraphs 
are separated by space.</p>

<p>Line breaks can be used<br>
to break text within<br>
the same paragraph.</p>

Text Formatting Elements

HTML provides several elements for formatting text:

HTML Code:

<!-- Bold and Strong -->
<b>Bold text (visual only)</b>
<strong>Strong text (important)</strong>

<!-- Italic and Emphasis -->
<i>Italic text (visual only)</i>
<em>Emphasized text (stressed)</em>

<!-- Other formatting -->
<u>Underlined text</u>
<s>Strikethrough text</s>
<mark>Highlighted text</mark>
<small>Small text</small>
<sub>Subscript</sub> and <sup>Superscript</sup>

Semantic vs Visual Elements

‚úÖ Semantic (Recommended)

  • <strong> - Important text
  • <em> - Emphasized text
  • <mark> - Highlighted text
  • <cite> - Citations
  • <code> - Code snippets

‚ö†Ô∏è Visual (Use with caution)

  • <b> - Bold appearance
  • <i> - Italic appearance
  • <u> - Underlined text
  • <s> - Strikethrough

Special Text Elements

HTML Code:

<!-- Quotations -->
<blockquote>
  This is a long quotation that stands alone as a block.
</blockquote>
<q>This is a short inline quotation.</q>

<!-- Code and preformatted text -->
<code>console.log("Hello World");</code>
<pre>
  This text preserves
    spaces and line breaks
      exactly as written.
</pre>

<!-- Abbreviations and definitions -->
<abbr title="HyperText Markup Language">HTML</abbr>
<dfn>HTML</dfn> is the standard markup language.

<article>

  <p>Web development is the process of creating websites and web applications.

  <h2>Key Technologies</h2>
  <p>The foundation of web development includes:</p>
  <p><abbr title="HyperText Markup Language">HTML</abbr> for structure,<br>

  <blockquote>
    "The best way to learn web development is by building projects."
  </blockquote>
</article>

Key Takeaways

  • Use headings (h1-h6) to create document structure
  • Paragraphs (<p>) group related sentences
  • Choose semantic elements over visual ones when possible
  • <strong> and <em> convey meaning, not just appearance
  • Use <code> for inline code and <pre> for code blocks
  • Blockquotes are for longer quotations, <q> for inline quotes
