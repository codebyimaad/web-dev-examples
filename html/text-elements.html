Introduction to Text Elements

Text is the foundation of web content. HTML provides various elements to structure and format text, from headings and paragraphs to emphasis and special formatting. Understanding these elements is crucial for creating well-structured, accessible web pages.

HTML Headings

HTML provides six levels of headings, from <h1> (most important) to <h6> (least important):

HTML Code:

<h1>Main Heading (Most Important)</h1>
<h2>Section Heading</h2>
<h3>Subsection Heading</h3>
<h4>Sub-subsection Heading</h4>
<h5>Minor Heading</h5>
<h6>Smallest Heading</h6>

  • Use headings for structure, not just styling

Paragraphs and Line Breaks

The <p> element defines paragraphs, while <br> creates line breaks:

HTML Code:

<p>This is the first paragraph. It contains multiple sentences 
that form a complete thought or idea.</p>

<p>This is the second paragraph. Notice how paragraphs 
are separated by space.</p>

Key Takeaways

  • Use headings (h1-h6) to create document structure
  • Paragraphs (<p>) group related sentences
  • Choose semantic elements over visual ones when possible
  • <strong> and <em> convey meaning, not just appearance
  • Use <code> for inline code and <pre> for code blocks
  • Blockquotes are for longer quotations, <q> for inline quotes
  • HTML entities display special characters safely (&lt;, &gt;, &amp;)
  • Use &nbsp; for non-breaking spaces and proper text spacing
  • Currency and symbol entities improve content readability
  • Unicode characters work directly with UTF-8 encoding
